🎭 CSRF Attack Visualizer

In a CSRF attack, the attacker crafts a malicious OAuth authorization URL using their own redirect_uri and tricks the victim into clicking it. The victim authenticates believing it is legitimate — but the authorization code is delivered to the attacker's server, not the real app. Without a state parameter, the client has no way to detect that it never initiated this flow.
